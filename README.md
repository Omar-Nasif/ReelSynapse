# ReelSynapse
ReelSynapse is an AI-driven inventory optimization and demand forecasting dashboard for cable manufacturing.  It predicts material requirements (copper, aluminum, PVC, XLPE, HDPE) using ML-based time-series analysis,  and provides real-time inventory insights, scenario simulation, and supply chain decision support.
